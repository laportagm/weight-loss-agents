# Next-Generation Weight-Loss Agents — Pharmacology Reference

A single-page, sourced reference comparing next-generation weight-loss agents by
receptor mechanism (the "four levers"), clinical efficacy, development stage,
real-world availability, and legitimate U.S. access & cost.

Educational reference — **not medical advice**. Pricing figures are approximate,
gathered June 2026 from manufacturer programs and FDA/NABP guidance, and change.

Open `index.html` (or the live GitHub Pages link). Installable: open in Safari →
**Add to Home Screen** to launch it fullscreen like an app.
